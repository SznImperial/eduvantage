# EduVantage 🎓

A modern, scalable multi-tenant school management system designed to streamline institutional operations, administrative tasks, and student data management. 

Built with performance and security in mind, EduVantage provides isolated environments for multiple schools under a single unified platform.

## 🚀 Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) 
*   **Backend & Database:** [Supabase](https://supabase.com/) (PostgreSQL)
*   **Authentication & Security:** Supabase Auth & Row-Level Security (RLS)
*   **Styling:** Minimalist, high-aesthetic UI (Tailwind CSS / Custom UI)

## ✨ Key Features

*   **Multi-Tenancy Architecture:** Secure, isolated data management for multiple schools utilizing Supabase RLS.
*   **Role-Based Access Control (RBAC):** Distinct permissions and customized dashboards for school admins, teachers, and students.
*   **Seamless Authentication:** Fast, secure, and reliable login flows.
*   **Premium User Experience:** A professional, dark-mode friendly (or easily adaptable) interface built for administrative efficiency without the clutter.
